_Experiment.Sample_label (SampleLabel)

(go to parent tag category listing)
NMR-STAR tag name_Experiment.Sample_label 
DescriptionNMR-STAR name: _Experiment.Sample_label
Enter the label assigned to one sample listed above in the section where samples and their contents were entered. A pull down list of sample labels should appear, if the samples have been defined above. If the same NMR experiment was carried out with more than one sample, enter a new row in the table for each time the NMR experiment was carried out. For example, if a 1H-15N HSQC experiment was run on an 15N labeled sample and a 15N-13C labeled sample, two rows should be entered one with the 15N sample indicated and one with the 15N-13C sample indicated.
